# SWAPIbox

A project designed around fetching STARWARS related data from multiple different sources and rendering the appropriate data upon user interaction.

## Installation

```
1. git clone
2. npm install
3. npm start
```

## Built with

```
- React
- JavaScript
- Postman
- Sass
- Node
```
